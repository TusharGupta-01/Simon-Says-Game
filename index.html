Simon Says Game

Press any key to the game

    How to Play
  1. Watch the Pattern: The game will light up a series of colors. Watch carefully!
  2. Repeat the Colors: When the pattern is done, tap the color buttons in the same order.
  3. Level Up: Get it right? The pattern gets longer each round!
  4. Miss a Step: Make a mistake, and the game ends. Try again to beat your best score!

Tip: Stay focused and see how many colors you can remember!
